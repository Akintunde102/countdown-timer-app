# CountDown Timer App


## Description
A Simple Count Down Android Timer App Built With React Native 

## Installation
```
npm install
```

### Generate An Android APK
```
npm run generate-apk
```

### Run in Emulator 
```
npm run android
```

### Testing

```
npm test
```
